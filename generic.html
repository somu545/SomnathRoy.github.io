Autonomous Car

This is an IOT based robo car Project which can be a self driving car or can be a bluetooth control car. An IoT-based Arduino car is a vehicle that incorporates Internet of Things (IoT) technology and is controlled using an Arduino microcontroller. Arduino is an open-source hardware and software platform that allows users to create interactive projects, including robotics and IoT applications..
